# 🧠 Multi-Agent Meeting Intelligence Assistant

A real-time, AI-powered meeting assistant built with TypeScript and multi-agent orchestration.

This system goes beyond traditional meeting tools by coordinating multiple AI agents to understand conversations, automate workflows, and turn meetings into actionable intelligence.

## 🚀 Overview

Meetings are one of the biggest productivity bottlenecks in organizations. Important decisions get lost, action items are missed, and follow-ups take time.

This project solves that by introducing a **multi-agent system** that:

- Listens to meetings in real time  
- Understands conversations and context  
- Answers questions during the meeting  
- Generates structured meeting minutes  
- Automatically sends follow-ups  
- Integrates with email and calendar systems  

---

## 🧩 Key Features

### 🎤 Real-Time Meeting Intelligence
- Joins meetings automatically

## 🤖 Multi-Agent Architecture

This system uses **agentic orchestration** to coordinate multiple specialized agents:

- **Orchestrator Agent**  
  Coordinates workflows and delegates tasks  

- **Meeting Intelligence Agent**  
  Handles real-time transcription and understanding  

- **Summarization Agent**  
  Generates structured meeting minutes  

- **Email Agent**  
  Sends automated follow-ups  

- **Productivity Agent**  
  Manages inbox and calendar interactions  

Each agent is responsible for a focused task, making the system scalable and reliable.

## ⚙️ How It Works

1. User schedules or joins a meeting  
2. Assistant joins and listens in real time  
3. Agents process and understand conversation context  
4. Users can query the meeting live  
5. After the meeting:
   - Summary is generated  
   - Action items are extracted  
   - Follow-up email is sent automatically
